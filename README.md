# Sistema de Gerenciamento de Ativos de TI

Projeto desenvolvido como Teste Técnico para a posição de Full-Stack Developer - Startup Academy.

**Desenvolvedor:** Italo Andrade De Sousa  
**Instituição:** UNICEPLAC (Engenharia de Software - 3º Semestre)

## Tecnologias Utilizadas
- **Backend:** Node.js, Express, PostgreSQL (Supabase)
- **Frontend:** React (Vite)
- **Banco de Dados:** PostgreSQL (Hospedado via Supabase)
